# 真話翻譯機

解碼人類的話中有話技術。

這是一個純前端單頁網站，可以直接用 GitHub Pages、Netlify、Vercel 或任何靜態網站服務部署。

## 功能

- 貼上公告、聲明稿、Email、顧問話或活動文案
- 輸出「說人話」、「話中有話」、「最想讓你相信什麼」、「鄉民翻譯」、「核彈翻譯」
- 支援複製單項、複製整段、原生分享
- 不需要後端，不會儲存使用者輸入

## 本機預覽

直接打開 `index.html` 即可。

也可以用本機伺服器：

```bash
python3 -m http.server 4188
```

然後打開：

```text
http://127.0.0.1:4188/
```

## 部署

把整個資料夾上傳到 GitHub，開啟 GitHub Pages，或把 `index.html` 拖到 Netlify Drop。


## 定位

真話翻譯機不是讀心術，也不是事實判決機。

它是一個語氣與語意拆解工具，用來幫助使用者看見一段話裡可能存在的包裝、暗示、避重就輕、情緒管理或公關語氣。

它適合用來分析：

- 公告
- 聲明稿
- Email
- 顧問式話術
- 活動文案
- 社群貼文
- 模糊、委婉、過度包裝的文字

## 使用方式

1. 打開 `index.html`
2. 貼上想分析的文字
3. 點擊分析 / 翻譯
4. 查看不同風格的輸出
5. 視需要複製或分享結果

## 輸出類型

### 說人話

把原文翻成比較直接、容易理解的版本。

### 話中有話

指出文字中可能隱含的暗示、避重就輕或未明說的部分。

### 最想讓你相信什麼

分析這段文字希望讀者接受的核心敘事。

### 鄉民翻譯

用比較犀利、網路語感的方式重寫。

### 核彈翻譯

用最不客氣的方式拆解，但仍應避免人身攻擊與無根據指控。

## 安全邊界

請注意：

- 這個工具只能分析語氣與可能意圖，不能證明對方真正想法。
- 輸出內容是推測，不是事實。
- 不應用來騷擾、霸凌、造謠或攻擊特定個人。
- 涉及法律、醫療、財務、人身安全或重大決策時，請不要只依賴本工具。
- 如果文字內容涉及具名個人，分享前請自行遮蔽敏感資訊。

一句話：

> 它可以幫你看懂話術，但不能替你審判人心。

## 隱私

目前版本為純前端工具，不需要後端，也不會主動儲存使用者輸入。

若未來加入 API、資料庫或登入功能，應重新補充隱私政策與資料處理說明。
